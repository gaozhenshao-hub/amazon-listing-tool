# 亚马逊高级A+模块指南 - 关键信息提取

## 通用规格
- 文件类型: .jpg, .bmp, .png (不支持GIF)
- 文件大小: 单个不超过2MB
- 分辨率: 至少72dpi
- 色彩空间: RGB (不支持CYMK)

## 模块列表（目录）
1. 高级完整图片
2. 高级文本
3. 高级图片和文本（包含文本的高级背景图像）
4. 高级视频
5. 高级比较表
6. 高级热点
7. 高级轮播
8. 高级问答
9. 高级技术规格
10. 品牌亮点

## 模块详细规格

### 1. 高级完整图片 (Premium Full Image)
- 桌面: 1464x600px (最小)
- 移动: 600x450px (最小)
- 标题: 80字符, 正文: 300字符
- 推荐用例: 产品特写、展示产品大小/尺寸或使用场景、品牌标识

### 2. 高级文本 (Premium Text)
- 标题: 80字符, 正文: 300字符
- 推荐用例: 用更多字符解释产品细节或说明

### 3. 包含文本的高级背景图像 (Premium Image with Text Overlay)
- 桌面: 1464x600px (最小)
- 移动: 600x450px (最小)
- 副标题: 40字符, 标题: 60字符, 正文: 300字符
- 文本框可左侧或右侧, 文本颜色黑色或白色
- 推荐用例: 产品特写、展示使用场景、突出关键产品特征/优势

### 4. 高级四图片和文本 (Premium Four Image and Text)
- 桌面: 300x225px (最小) x4张
- 移动: 无-利用桌面版图像素材
- 副标题: 80字符, 标题: 30字符, 正文: 150字符
- 推荐用例: 产品各角度照片、展示不同用途、展示不同特性

### 5. 包含文本的高级双图片 (Premium Dual Image with Text)
- 桌面: 650x350px (最小) x2张
- 移动: 无-利用桌面版图像素材
- 副标题: 50字符, 标题: 50字符, 正文: 300字符
- 推荐用例: 产品各角度照片、展示不同用途、展示不同特性

### 6. 带文本的单张高级图片 (Premium Single Image with Text)
- 桌面: 800x600px (最小)
- 移动: 无-利用桌面版图像素材
- 副标题: 40字符, 标题: 80字符, 正文: 500字符
- 推荐用例: 产品特写、展示产品大小/尺寸、突出关键产品特征/优势

### 7. 高级全视频 (Premium Full Video)
- 视频类型: 仅限.mp4
- 视频分辨率: 960:540 (最小)
- 视频大小: 200MB (最大), 视频长度: 180秒 (最大)
- 图像(视频预览): 1464x600px (最小)
- 标题: 80字符, 正文: 300字符
- 推荐用例: 展示产品移动和操作、组装产品、不同用途

### 8. 包含文本的高级视频 (Premium Video with Text)
- 视频: 800x600px (最小), 200MB (最大), 180秒 (最大)
- 图像(视频预览): 800x600px (最小)
- 副标题: 40字符, 标题: 80字符, 正文: 500字符
- 视频标题: 50字符, 视频说明: 200字符
- 推荐用例: 展示产品移动和操作、组装产品、提供背景信息

### 9. 高级比较表1 (Premium Comparison Table 1)
- 图像: 200x225px (最小)
- 模块标题: 80字符, 图像标题: 25字符, 特征: 30字符, 工具提示: 150字符, 详细信息: 30字符
- 产品: 最少4个-最多7个, 特征: 最少5个-最多12个
- 推荐用例: 显示不同产品之间的特征或用例差异、规格差异、追加销售

### 10. 高级比较表2 (Premium Comparison Table 2)
- 图像: 300x225px (最小)
- 模块标题: 80字符, 图像标题: 30字符, 特征: 30字符, 正文: 80字符
- 产品: 最少2个-最多3个, 特征: 最少2个-最多5个
- 推荐用例: 显示不同产品之间的特征或用例差异、规格差异

### 11. 高级比较表3 (Premium Comparison Table 3)
- 图像: 488x700px (最小), 移动: 600x450px (最小)
- 图表标题: 25字符, 图像标题: 25字符, 特征: 20字符, 特征文本: 20字符
- 比较产品: 最少2个-最多4个, 特征: 最少3个-最多7个
- 推荐用例: 显示不同产品之间的特征或用例差异、规格差异

### 12. 高级热点1 (Premium Hotspot 1)
- 图像: 1464x600px (最小), 移动: 600x450px (最小)
- 热点标题: 50字符, 热点正文: 200字符
- 热点: 最少2个-最多6个
- 推荐用例: 为复杂功能繁多的产品简化并搭建特点/优势的标注

### 13. 高级热点2 (Premium Hotspot 2)
- 图像(用于视频预览): 1464x600px (最小), 移动: 600x450px (最小)
- 模块标题: 80字符, 正文: 300字符, 热点标题: 50字符
- 热点: 最少2个-最多6个
- 推荐用例: 简化和构建复杂、特征繁多的产品的特征/优势标注

### 14. 高级导航轮播 (Premium Navigation Carousel)
- 桌面: 1464x600px (最小), 移动: 600x450px (最小)
- 导航文本: 25字符, 副标题: 25字符, 标题: 25字符, 正文: 100字符
- 面板: 最少2个-最多5个
- 推荐用例: 显示目录中产品特征差异、追加销售、产品各角度照片、展示不同用途

### 15. 高级规则轮播 (Premium Guided Carousel)
- 桌面: 1464x600px (最小), 移动: 600x450px (最小)
- 模块标题: 100字符, 插入标题文本: 20字符, 插入正文: 100字符, 导航文本: 20字符
- 面板: 最少2个-最多5个
- 推荐用例: 显示目录中产品特征差异、追加销售、产品各角度照片、展示不同用途、突出使用产品关键步骤

### 16. 高级简单图像轮播 (Premium Simple Image Carousel)
- 桌面: 1464x600px (最小), 移动: 600x450px (最小)
- 标题文本: 50字符, 正文: 200字符
- 面板: 最少2个-最多6个
- 推荐用例: 产品特写、展示产品大小/尺寸、突出关键产品特征/优势

### 17. 高级视频图像轮播 (Premium Video Image Carousel)
- 视频: 800x600px (最小), 200MB (最大), 180秒 (最大)
- 图像: 800x600px (最小)
- 标题: 80字符, 面板标题: 50字符, 面板副标题: 80字符, 面板正文: 500字符
- 视频标题: 50字符, 视频说明: 200字符
- 面板: 最少2个-最多6个
- 推荐用例: 不同方面内容的短视频剪辑

### 18. 高级问答 (Premium Q&A)
- 图像: 1464x600px (最小), 移动: 无-利用桌面版素材
- 问答限制: 最少2个-最多5个
- 问题文本: 120字符, 回答文本: 250字符
- 推荐用例: 主动回答常见问题、解决容易导致客户投诉或退货的常见问题

### 19. 高级技术规格 (Premium Technical Specifications)
- 图像: 300x300px (最小), 移动: 无-利用桌面版素材
- 标题: 80字符, 规格名称: 30字符, 规格值: 60字符
- 规格: 最少3个-最多15个
- 推荐用例: 展示产品的技术规格

### 20. 品牌亮点 (Brand Highlights)
- 图像: 135x135px (最小)
- 标题: 30字符, 正文: 80字符
- 亮点: 最少3个-最多4个
- 推荐用例: 品牌故事、品牌价值观
