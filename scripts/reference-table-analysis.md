# 参考表格结构分析

## 表格列分组

### 产品信息
- 产品序号
- 产品名称
- 店铺
- SKU
- 上架时间
- 品类/品牌信息
- 上架时间/产品信息

### 产品利润情况 (字段列)

- 毛利润: 58.91 (绿色高亮)
- 毛利率: 29.46%
- 退货率: 0.00%
- 评分: 3.5/14

### 总体情况 (每周一行)
- 数据: 月度汇总标题行 (如 "9月度汇总")
- 时间: 具体日期 (如 9月5日, 9月12日...)
- 销量趋势: 平稳/上升/下降
- 销量: 数字
- 订单量: 数字
- 销售额: $金额
- 订单利润: $金额 (负数红色括号)
- 订单利润率: 百分比

### 广告情况
- 毛利毛利率%: 百分比
- Session-total: 数字
- 化率(CV): 百分比 (总转化率)
- 广告CVR: 百分比
- 自然CVR: 百分比
- 广告订单: 数字
- 自订单(自然订单): 数字
- 广告点击: 数字
- 自然点击率(CTR): 百分比
- 广告曝光%: 百分比
- CPC: 金额
- 广告花费: 金额
- ACOS: 百分比

### 产品质量情况
- 评分: 数字
- 评论数: 数字
- 退货率%: 百分比

## 数据展示方式
1. 每个产品按父ASIN为一个区块
2. 每个区块内：
   - 产品基本信息行（售价、平手价、毛利润等）
   - 按月分组，每月有月度汇总行（绿色背景）
   - 每周一行数据
3. 显示最近四周数据
4. 月度汇总行显示：务实际利润、订单利润额
