# 广告模块数据上传方案

## 一、现状分析

### 当前领星API调用清单

广告模块目前通过领星ERP API获取以下数据，共涉及 **14个API接口**：

| 序号 | API路径 | 数据类型 | 用途 |
|------|---------|---------|------|
| 1 | `/pb/openapi/newad/portfolios` | 广告组合 | 获取广告组合列表（名称、预算、状态） |
| 2 | `/pb/openapi/newad/spCampaigns` | SP广告活动 | 获取SP广告活动列表 |
| 3 | `/pb/openapi/newad/hsaCampaigns` | SB广告活动 | 获取SB广告活动列表 |
| 4 | `/pb/openapi/newad/sdCampaigns` | SD广告活动 | 获取SD广告活动列表 |
| 5 | `/pb/openapi/newad/queryWordReports` | SP搜索词报告 | 搜索词12分类分析 |
| 6 | `/pb/openapi/newad/hsaQueryWordReports` | SB搜索词报告 | SB搜索词分析 |
| 7 | `/pb/openapi/newad/spSearchTermReports` | SP搜索词详情 | 搜索词趋势对比 |
| 8 | `/pb/openapi/newad/spKeywordReports` | SP关键词报告 | 投放对象分析、关键词表现 |
| 9 | `/pb/openapi/newad/hsaKeywordReports` | SB关键词报告 | SB投放对象分析 |
| 10 | `/pb/openapi/newad/campaignPlacementReports` | SP广告位报告 | 广告位分析（首页/商品页/其他） |
| 11 | `/pb/openapi/newad/hsaCampaignPlacementReports` | SB广告位报告 | SB广告位分析 |
| 12 | `/pb/openapi/newad/sdCampaignReports` | SD广告活动报告 | SD广告位分析 |
| 13 | `/pb/openapi/newad/spCampaignHourData` | SP分时数据 | 分时竞价策略 |
| 14 | `/pb/openapi/newad/sbCampaignHourData` / `sdCampaignHourData` | SB/SD分时数据 | SB/SD分时分析 |
| 15 | `/pb/openapi/newad/spProductAds` / `sdProductAds` | 商品广告 | ASIN-广告活动映射 |
| 16 | `/pb/openapi/newad/sdMatchTargetReports` | SD投放目标报告 | SD投放对象分析 |
| 17 | `/pb/openapi/newad/listHsaTargetingReport` | SB投放目标报告 | SB投放对象分析 |
| 18 | `/erp/sc/data/seller/lists` | 店铺列表 | 获取SID |
| 19 | `/erp/sc/data/mws/listing` | 产品列表 | 获取ASIN列表 |
| 20 | `/erp/sp/query/queryUserSearchTerm` | 搜索词排名 | 词频分析 |
| 21 | `/erp/sp/data/getKeywordsReports` | 关键词报告 | 有效词分析 |

### 现有上传功能

系统已有 **AdReportImportTab** 组件，支持上传领星导出的广告报告Excel，解析后存入 `ad_keyword_weekly` 表。但该功能仅覆盖关键词周度数据，不包含广告活动结构、分时数据等。

## 二、表格上传方案

### 数据分为两大来源

#### A. 领星ERP后台导出（推荐）

领星后台 → 广告管理 → 报表下载，可以导出以下报告：

| 报告名称 | 下载路径 | 覆盖功能 | 上传频率 |
|---------|---------|---------|---------|
| **关键词报告** | 领星 → 广告管理 → 报表 → 关键词报告 | 搜索词12分类、投放对象分析、词频分析、有效词分析 | 每周一次 |
| **搜索词报告** | 领星 → 广告管理 → 报表 → 搜索词报告 | 搜索词分析、趋势对比、否定词建议 | 每周一次 |
| **广告位报告** | 领星 → 广告管理 → 报表 → 广告位报告 | 广告位分析（首页/商品页/其他） | 每周一次 |
| **广告活动报告** | 领星 → 广告管理 → 报表 → 广告活动报告 | 预算分配、总览数据 | 每周一次 |

#### B. 亚马逊广告后台导出

亚马逊广告控制台 → 报告 → 创建报告，可以导出更原始的数据：

| 报告名称 | 下载路径 | 覆盖功能 | 上传频率 |
|---------|---------|---------|---------|
| **SP搜索词报告** | 亚马逊广告 → 报告 → SP → 搜索词 | 搜索词12分类、趋势对比 | 每周一次 |
| **SP投放报告** | 亚马逊广告 → 报告 → SP → 投放 | 投放对象分析、关键词表现 | 每周一次 |
| **SP广告位报告** | 亚马逊广告 → 报告 → SP → 广告位 | 广告位分析 | 每周一次 |
| **SP广告活动报告** | 亚马逊广告 → 报告 → SP → 广告活动 | 预算分配、总览 | 每周一次 |
| **SB搜索词报告** | 亚马逊广告 → 报告 → SB → 搜索词 | SB搜索词分析 | 每周一次 |
| **SD投放报告** | 亚马逊广告 → 报告 → SD → 投放 | SD投放对象分析 | 每周一次 |

### 分时数据特殊处理

**分时竞价策略**（hourly data）是领星API独有的数据维度，亚马逊后台和领星后台均不提供分时维度的报告下载。

**方案选择：**
- **方案A（推荐）**：保留领星分时API调用（仅此一个接口），其他全部切换为表格上传
- **方案B**：放弃分时策略功能，全部切换为表格上传
- **方案C**：手动录入分时数据（工作量大，不推荐）

## 三、推荐最终方案

### 需要上传的表格清单（共4-6个文件）

| 序号 | 表格名称 | 来源 | 包含字段 | 说明 |
|------|---------|------|---------|------|
| 1 | **搜索词报告** | 领星导出 或 亚马逊SP/SB | 搜索词、广告活动、广告组、曝光、点击、花费、销售额、订单、ACoS等 | 核心数据，覆盖搜索词分析、12分类、趋势对比 |
| 2 | **关键词/投放报告** | 领星导出 或 亚马逊SP/SB/SD | 关键词、匹配类型、出价、曝光、点击、花费、销售额、订单等 | 覆盖投放对象分析、有效词、词频分析 |
| 3 | **广告位报告** | 领星导出 或 亚马逊SP | 广告位类型、曝光、点击、花费、销售额、ACoS等 | 覆盖广告位分析 |
| 4 | **广告活动报告** | 领星导出 或 亚马逊SP/SB/SD | 广告活动名称、类型、预算、状态、花费、销售额等 | 覆盖总览、预算分配 |
| 5 | **广告组合映射表** | 系统生成模板 | 广告组合名称 → 父ASIN映射 | 已有功能，用于将广告数据关联到产品 |
| 6 | **（可选）ASIN广告报告** | 亚马逊SP/SD | 推广的ASIN、广告活动、曝光、点击、花费等 | 覆盖ASIN汇总功能 |

### 上传工作流

1. **首次设置**：下载广告组合映射模板 → 填写父ASIN → 上传映射表
2. **每周上传**：从领星/亚马逊下载4个报告 → 选择周期 → 批量上传 → 系统自动解析并关联到产品
3. **数据分析**：上传完成后，所有分析Tab自动从本地数据库读取，不再调用领星API

### 保留的领星API（可选）

- 分时数据API（`spCampaignHourData` / `sbCampaignHourData` / `sdCampaignHourData`）：如果需要分时策略功能
- 店铺列表API（`/erp/sc/data/seller/lists`）：用于获取SID，但也可以通过上传时手动选择店铺替代

## 四、实施优先级

1. **Phase 1**：搜索词报告上传 → 覆盖搜索词12分类、趋势对比、否定词建议
2. **Phase 2**：关键词/投放报告上传 → 覆盖投放对象分析、有效词、词频分析
3. **Phase 3**：广告活动报告上传 → 覆盖总览、预算分配
4. **Phase 4**：广告位报告上传 → 覆盖广告位分析
5. **Phase 5**：ASIN广告报告上传 → 覆盖ASIN汇总

每个Phase完成后，对应的分析Tab从"调用领星API"切换为"读取本地数据库"。
